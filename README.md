## 📋 Visão Geral
Solução técnica desenvolvida como resposta a teste técnico, demonstrando:

✅ Filtros combinados (AND/OR)  
✅ Validação de formulários com Zod  
✅ Mock API integrado  
✅ UI acessível com Radix UI  
✅ Gerenciamento de estado com Zustand
